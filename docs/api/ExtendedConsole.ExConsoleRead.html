Parameters

self ExConsole

The error message to show the user if the conversion failed.

converter Func<string, (bool Success, T Value)>

A function that takes in a string and returns a ValueTuple of bool success and T value.

Returns

T

The T value converted from the input string, or null if the user entered ^Z.

Type Parameters

T

The target type of the conversion.

Examples

Parse and get an array of int values from comma-separated integers entered by the user.

var ints = exConsole.ReadClass(
    "Please enter a list of comma-separated integers",
    "Invalid value entered",
    str => {
        var arr = str.Split(',');
        if (arr.All(s => int.TryParse(s, out var i)))
        {
            return (true, arr.Select(s => int.Parse(s)).ToArray());
        }
        return (false, default(int[]));
    }
);
