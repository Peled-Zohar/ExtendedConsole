Examples

Create and run a menu with the specified title and items. This menu does not execute anything, it only returns the zero-based index of the item selected by the user

    "What to do next?",
    false,
    "<c f='Yellow'>quit<c>",
    "Do this",
    "Do that"
);
 // Do something with the selection here.
