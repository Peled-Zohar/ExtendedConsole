Namespace ExtendedConsole

Classes

ExConsole

Provides support for using an xml based markup text in console applications.

Markup contains text and control tags to set the foreground and background color of the text. Color names are limited to members of the ConsoleColor enum, but are case insensitive.

Markup example: 

<c f='Yellow'>yellow text<c b='blue'> on blue background.</c></c>
